# Changelog

## [1.0.0] - 2026-03-22
This is the very first version of this extension.

### Functionalities
- Commits the changes made after certain checkpoints during the implementation (such as completing a phase or a task within a phase)
- Commits the changes made after every pre-implementation step

### Added
- ```speckit.checkpoint.commit``` - Performs both of the funcionalities above
